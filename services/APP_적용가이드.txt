이번에 app.py 전체를 다시 덮어쓰기보다,
안전하게 아래만 반영하는 방식을 추천합니다.

1. services/naver_datalab.py 파일을 추가
2. 기존 app.py 상단 import에 추가
   from services.naver_datalab import search_keyword_trend, summarize_trend_result

3. 1단계 입력 화면의 '키워드 검색' 영역 버튼에서 아래 호출 사용

sub_kw_list = [x.strip() for x in st.session_state["sub_keywords"].split(",") if x.strip()]
result = search_keyword_trend(
    main_keyword=st.session_state["keyword"],
    sub_keywords=sub_kw_list,
    start_date="2025-01-01",
    end_date="2025-03-31",
    time_unit="month",
)
summary = summarize_trend_result(result)
st.session_state["keyword_trend_result_text"] = summary
st.rerun()

4. 세션 기본값에 추가
"keyword_trend_result_text": ""

5. 결과 표시용 text_area 추가
st.text_area(
    "키워드 검색 결과",
    value=st.session_state["keyword_trend_result_text"],
    height=250,
    disabled=True,
    label_visibility="collapsed",
    key="keyword_search_result_box",
)

기존 기능을 최대한 안 건드리려면 이 방식이 가장 안전합니다.
